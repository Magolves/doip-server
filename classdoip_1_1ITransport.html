Detailed Description

Transport layer abstraction for DoIP connections.

This interface abstracts the underlying transport mechanism (TCP, mock, etc.) from the DoIP protocol state machine. It provides methods for sending/receiving DoIP messages and managing the transport connection lifecycle.
