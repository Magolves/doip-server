13  * 3. Ignore SIGHUP
14  * 4. Second fork() - ensure daemon can't reacquire controlling terminal
15  * 5. Change working directory to /
16  * 6. Set file creation mask (umask)
17  * 7. Close all file descriptors
18  * 8. Redirect stdin/stdout/stderr to /dev/null
19  * 9. Write PID file (if specified)
20  *
21  * CRITICAL: This must be called BEFORE:
22  * - Creating any server objects
23  * - Opening sockets
24  * - Spawning threads
25  * - Initializing logging (switch to syslog AFTER daemonization)
26  * - Allocating significant resources
27  *
28  * @param pidfile Optional path to PID file (e.g., "/var/run/doipd.pid")
29  * @return true in child process on success, false on error
30  * (parent process exits successfully and never returns)
31  *
32  * @note After successful daemonization:
33  * - Parent process has exited
34  * - Child process has no controlling terminal
35  * - stdin/stdout/stderr point to /dev/null
36  * - Working directory is /
37  * - Process is session leader
38  *
39  * Example usage:
40  * @code
41  * if (daemon_mode) {
42  * if (!daemon::daemonize("/var/run/myapp.pid")) {
43  * std::cerr << "Daemonization failed" << std::endl;
44  * return 1;
45  * }
